How to use (GitHub Pages, JSON PWA)

  1. Create a new public repo.
  2. Upload the contents of this ZIP (not the ZIP itself).
  3. Enable GitHub Pages: Settings → Pages → Branch: main (root).
  4. Open the site, then use the in-app Update app button after future updates (forces new SW).
  5. Edit workouts.json to update content—no redeploy needed.
  6. Upload videos to /media with the exact filenames already referenced.
